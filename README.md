# Authorization and Authentication playground

This project aims to teach about authorization and authentication. It's about getting a better picture, materializing
it, and visualizing it. Eventually, to have something practical on hand when talking to the community or others.

The playground provides a hands-on way to learn AuthZ and AuthN specifications without vendor bias.

The primary aim is to gain a detailed understanding of these concepts. Prioritizing code quality or best practices will
come later.
